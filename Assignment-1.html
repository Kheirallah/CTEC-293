Answers

  2. If a doctype is not specified the webpage may be rendered in older versions of HTML which may not support the elements used within your code. For example, HTML5 tags such as footer may not function. The difference between quirks mode and standards mode is that quirks mode is used to render old-school webpages while standards mode renders webpages in modern form (Exact differences depend on the browser as they vary).
    • HTML 4.01 (Strict) doctype: <!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01//EN" "http://www.w3.org/TR/html4/strict.dtd">
    • HTML5 doctype: <!DOCTYPE HTML>
    • XHTML doctype: <!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Strict//EN" "http://www.w3.org/TR/xhtml1/DTD/xhtml1-strict.dtd">
  3. All of these browsers support just about all versions of HTML except for versions of Internet Explorer 8 & below which have issues with identifying HTML5 elements.
